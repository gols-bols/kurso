# ИС управления заявками (рабочий сайт на PHP)

Это рабочая серверная часть (с веб-интерфейсом) для системного администратора ГБПОУ МО СПК.

## Возможности
- Регистрация и вход пользователей.
- Роли пользователей (`user`, `admin`, `manager`).
- Создание и просмотр заявок.
- Смена статусов заявок (для администратора/менеджера).
- Комментарии к заявкам.
- Автоматическая инициализация SQLite-базы при первом запуске.

## Быстрый запуск
```bash
php -S 127.0.0.1:8000 -t public
```

Открыть в браузере: `http://127.0.0.1:8000`.

## Тестовый админ
- Email: `admin@spk.local`
- Пароль: `admin123`

## Структура
- `public/index.php` — веб-приложение и маршрутизация.
- `src/bootstrap.php` — запуск сессии, подключение БД, helper-функции.
- `src/schema.sql` — схема базы данных.
- `data/app.sqlite` — SQLite-файл (создается автоматически).
