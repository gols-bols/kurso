# Макет для Figma

Для быстрого переноса в Figma подготовлен общий board:

- [spk_helpdesk_figma_board.svg](/Users/gnome/Downloads/kurso-main%202/docs/spk_helpdesk_figma_board.svg)

Что внутри:

- экран авторизации;
- экран журнала заявок;
- экран создания заявки;
- экран карточки заявки.

Как использовать:

1. Открыть Figma.
2. Перетащить `spk_helpdesk_figma_board.svg` на рабочую область.
3. При необходимости разгруппировать элементы и доработать текст, цвета и отступы.

Макет собран в той же логике, что и текущий интерфейс проекта: теплая светлая палитра, акцентные статусные цвета, карточная композиция и явное разделение ролей.
